# ECS Fargate VPC Platform

A containerised REST API running on AWS ECS Fargate inside a custom VPC, behind
an Application Load Balancer, backed by RDS PostgreSQL — all defined in
Terraform, with a CI pipeline and a read-only cost-audit tool.

The application itself is deliberately small (a health check and a two-endpoint
items API). The focus of the project is the infrastructure around it:
networking, security-group layering, secrets handling, containers, and
deployment.

## Architecture

```
Internet
   │
   ▼
Application Load Balancer            (public subnets, 2 AZs)
   │  HTTP :80  ──► target group ──► /health
   ▼
ECS Fargate service (2 tasks)        (private subnets, no public IP)
   │  container :8000
   ▼
RDS PostgreSQL                       (private subnets, encrypted, not public)

Supporting: single NAT gateway (egress for tasks), Secrets Manager (DB creds),
KMS (RDS encryption), CloudWatch (logs + 3 alarms), VPC flow logs, ECR.
```

Security groups are chained so each tier only accepts traffic from the one in
front of it: internet → ALB, ALB → ECS, ECS → RDS. The database is not
reachable from the internet at all.

## Quick start (local, no AWS)

```bash
# App tests
cd app
pip install fastapi "uvicorn[standard]" pydantic pytest httpx
python -m pytest tests -q          # 8 passed

# Run the API locally
docker build -t ecs-app:local .
docker run -d -p 8000:8000 -e SKIP_DB_INIT=1 --name ecs-app ecs-app:local
curl http://localhost:8000/health  # {"status":"ok"}
docker stop ecs-app && docker rm ecs-app

# Terraform
cd ../terraform
terraform init -backend=false
terraform validate                 # Success!

# Cost-audit tests
cd ..
python -m pytest cost-audit/tests -q   # 10 passed
```

## Design choices and trade-offs

Documented in [DECISION_RECORD.md](./DECISION_RECORD.md). In short: Fargate over
EKS/EC2 for one small service; single NAT gateway and single-AZ RDS as
deliberate cost economies (with the production alternatives noted); three-tier
security groups; DB credentials generated by Terraform and stored in Secrets
Manager, never hardcoded; split IAM execution/task roles for least privilege.

## Cost

Roughly $80–100/month if left running — NAT gateway, ALB, Fargate, and RDS are
the main drivers. The cost-audit tool surfaces exactly these. For a portfolio,
deploy-verify-destroy in one sitting keeps it to a few dollars; the teardown
steps are in [DEPLOYMENT.md](./DEPLOYMENT.md).
